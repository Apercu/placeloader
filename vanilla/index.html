install & usage

Import the style the way you prefer, either using the source scss file or the compiled and minified css. 
@import '~placeloader/main.scss'
<link rel="stylesheet" href="https://unpkg.com/react-placeloader/dist/main.css">
And then simply declare the placeloader class you want on an element. Keep in mind the speed of the animation will differ depending on its size, as shown in the below examples.
